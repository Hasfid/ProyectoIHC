# Skill Registry - ProyectoIHC

## Project Standards

### TypeScript & React Native
- Use functional components and hooks.
- Use TypeScript for all new code.
- Ensure SSR safety for web (check for `window` availability).
- Use `expo-router` for navigation.

### Supabase
- Use the shared client in `lib/supabase.ts`.
- Handle errors gracefully using `Alert.alert`.

### Web Compatibility
- Use `.web.tsx` for platform-specific web implementations if needed.
- Ensure components are responsive.
